# ArToo Campus
MEAN exercises, examples and project for ArToo Campus.

## Start
Fork this repo.

## Setup git
```
git init
git config --global pull.rebase true
git clone git@github.com:yourUsername/artoo-campus.git .
```

## Install packages
```
npm install
```

## Install gulp
```
(sudo) npm install -g gulp
```
